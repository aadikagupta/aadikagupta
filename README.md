## 📌 Projects

### 🎛️ MIDIfy — DIY MIDI DJ Controller
Built a custom MIDI DJ controller using Arduino Uno and repurposed HDD components to simulate vinyl scratching. Integrated potentiometers, sliders, and buttons for real-time control mapped in Ableton Live 11.
`Arduino` `C++` `MIDI` `Embedded Systems` `Hardware`

### 🖼️ Image Processing — Colour Change & Rotation
Developed an image processing pipeline in GNU Octave to convert RGB images to grayscale and perform 90° rotation using matrix manipulation and DSP fundamentals.
`GNU Octave` `DSP` `Image Processing` `Signal Processing`

### 📐 Tilt Detection using ADXL335 Accelerometer
Designed an analog-to-digital tilt detection system using the ADXL335 3-axis accelerometer and op-amp comparator circuits (IC741), producing 2-bit digital outputs with LED indicators.
`Circuit Design` `Analog Electronics` `Arduino` `Sensors`
